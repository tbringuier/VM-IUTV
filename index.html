VM-IUTV

Machine virtuelle pédagogique compatible VirtualBox & VMWare pour le BUT Réseaux et Télécommunications !

Configuration recommandée

RAM 4 Go
Stockage 16 Go
CPU 4 Coeurs

Basée sur Lubuntu 24.04 elle inclut les outils suivants :

  • GNS3
  • Marionnet
  • Thonny
  • VS Code
  • PHP
  • Apache2
  • PostgreSQL
